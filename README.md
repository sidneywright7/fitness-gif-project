## Overview
This fitness GIF project is a fitness demonstration search engine that allows users to search for workout movements and view animated GIFs. 

## How to Use
1. Enter an exercise name in the search bar (e.g., "squat", "plank", "burpee")
2. Click the Search button or press Enter
3. Browse through the exercise demonstration GIFs
4. Try different search terms to discover new movements

## Technologies Used
- HTML
- CSS
- JavaScript
- Giphy API
- Flexbox & Custom Grid System
- Google Fonts
- GitHub Pages (hosting)
